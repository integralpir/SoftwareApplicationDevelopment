Рассмотрим более продвинутые программные средства. 
Максимально приблизимся к серьезной разработке Java + Spring на знакомом примере.

- что такое ORM;
- современный асинхронный клиент для сетевого взаимодействия webClient.

### Занятие №2

Реализовали:
